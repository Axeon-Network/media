**Cuber build 64** is the second known build of *[Project Cuber](Cuber)*. It was compiled on 10 February 2026 at 3:51 pm from the decoy 26-02-10_kit branch.

This build introduces many major elements of the Watercolor CSS Framework (codenamed *Wh-Sheet*) used in *Cuber*, like windows and buttons, although window buttons are yet to be correctly implemented.

Additionally, the font was changed to 8pt Tahoma and the background color was changed to `#004E98`, coinciding with the background color of [Beta 1 builds of Windows XP](https://betawiki.net/wiki/Windows_XP#Beta_1). The build watermark was also updated to include the directory, hostname and port where Cuber is served to. This would later be scrapped in [build 351](Cuber_build_351).
