**Deltari build 4500** is a Release Candidate 1 build of Deltari, the underlying engine behind [KuroWiki](KuroWiki), compiled on 13 January 2026 at 18:20 UTC+1.

This build is notable for being included as part of the initial commit of the GitHub repository of [Deltari Meta](Deltari_Meta), meant to be a "reference implementation" and documentation source for Deltari.

This build is also notable for being the first `main` branch compile that is not associated with any public release since [the Pre-Alpha development stage of Deltari](Development_of_Deltari). A spin off from the `kayaurora` development branch of KuroWiki, it includes several unfinished RC1 features like the Download Box, but is missing other work from the `kitsixtyfour` branch, such as ModularDelta, which would only have an appearance in [build 4584](Deltari_build_4584).
