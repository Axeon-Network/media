***Stuffy's Website*** (commonly reffered to as ***LM Stuffy Website***[^1] in order to differenciate it from *[Stuffy Website WOU](Stuffy's_Website)*) was a website initially planned to serve as [KitSixtyFour](KitSixtyFour)'s personal website. Initial development can be traced back to February 2024, but most of it took place in April-May 2024. Work was finished but the website became lost media before it was even published, but a demo video was uploaded to [StuffyXP Extras](StuffyXP).

# Layout
LM Stuffy Website uses a basic layout with a simple text-only navigation bar on the top to various pages. The website uses Bulma as the CSS framework, with some of the elements inspired from [KayAurora](KayAurora)'s *[LynxWebsite](LynxWebsite)*.

Essentially speaking, the layout is similar to late 2024's *[StuffyXP Website](StuffyXP_Website)* except more barebones.

# History
Throughout development, it was initially meant for the website and blog to be separate, with the blog side powered by the Jekyll static site generator. A Software Repository was also planned but later cancelled during development due to lack of resources.

The website was KitSixtyFour's first webdev related project, and what they learned from this project was later used in *[Stuffy Website WOU](Stuffy's_Website)* and later projects.

## Becoming lost media
In late May 2024, this website's source code (along with one *[Stuffy's Tech Misadventures](Stuffy's_Tech_Misadventures)* episode and other files) have became lost media, having been backed up on a DVD which ended up unrecognized by Windows 7's File Explorer, as Windows Vista's File Explorer (where it was backed up from) did not successfully write the files to the disc.

It is notable to know that parts of the code have been sent in various Discord Servers KitSixtyFour was in, including their own [Stuffy's Place](Stuffy's_Place), in order to get the help building this website. Based on this code, and public screenshots/footage, the website could be theoretically reconstructed.
