This article shows a list of codenames used by [AveryEclipse](AveryEclipse) in their projects.

[C](#C) · [L](#L) · [P](#P) · [R](#R) · [S](#S) · [U](#U) · [V](#V)

# C
---

**Cosmos**
&nbsp;&nbsp;&nbsp;&nbsp;Codename for a 2018 revamp of the [Axeon Network](Axeon_Network#Discord_Server) Discord server.

# L
--- 

**Lyra**
&nbsp;&nbsp;&nbsp;&nbsp;Codename for the Discord server that would have resulted from the merge between [LynxPlaza](Axeon_Network) and [HoriServer (2)](HoriServer_(2)) following the general sunsetting of the [Lynxmic](Lynxmic) project in April 2025. Later cancelled as a result of the [HoriServer (2) incident](April_2025_HoriServer_(2)_Incident).

# P
---

**Photon**
&nbsp;&nbsp;&nbsp;&nbsp;Cancelled tech-oriented YouTube channel planned throughout February 2025.

**PreventeDev**
&nbsp;&nbsp;&nbsp;&nbsp;Preliminary name for *Whirlpool Bot Plaza*, see [its section](#W)

**Preventemod**
&nbsp;&nbsp;&nbsp;&nbsp;Name used for a project for a separate moderation bot for Discord that has been worked on in early 2018, later cancelled.
The name can be understood as sort of a combination between the words "prevent" ("*to prevent [bad event]*") and "mod" ("*to moderate - take action to stop/prevent [bad event]*"), with the letter "*e*" added between them in an attempt to create a more unique, fluent name.

# R
---

**Rano**
&nbsp;&nbsp;&nbsp;&nbsp;Name for an hypotethical operating system planned in 2017.

**Ruby**
&nbsp;&nbsp;&nbsp;&nbsp;Name used for a project for a separate fun-oriented bot for Discord from 2018-19, later cancelled.
Ruby's bot account still exists to this day and is currently used for [Orchid](Orchid)'s Discord side.

# S
---

**Softdron**
&nbsp;&nbsp;&nbsp;&nbsp;Name of a fictional company thought to have been the maker of the hypotethical [Rano](#R) operating system.

# U
---

**Uranus**
&nbsp;&nbsp;&nbsp;&nbsp;Cancelled replacement for [Sapphire](Sapphire).

# W
---

**Whirlpool**
Name used for the *Whirlpool Bot Plaza* Discord Server in November 2018, which was gonna be an unified place for support for both [Venus](Venus) and [Uranus](Sapphire) before the latter project was cancelled.
