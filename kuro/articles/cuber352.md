**Cuber build 352** is an Alpha build of *[Project Cuber](Cuber)*. It was compiled on 2 May 2026 at 8:52 am. Although the rest of the site isn't shown, this build's watermark was shown, using a watermark style akin to the one present on [Beta 2 builds of Windows XP](https://betawiki.net/wiki/Windows_XP#Beta_2) (that is, with Trebuchet MS used on the version banner alongside increased padding). However, this was only used for testing purposes and was never intended to be used in *Cuber* at all.
