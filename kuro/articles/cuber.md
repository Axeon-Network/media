***Project Cuber***, more commonly known as ***Cuber***, is the codename for a future currently work-in-progress website intended to serve as the future personal website of [KitSixtyFour](KitSixtyFour). It uses a design based on the *[Watercolor](https://betawiki.net/wiki/Watercolor)* visual style found in pre-release builds of Microsoft's [Windows XP](https://betawiki.net/wiki/Windows_XP) (codenamed *Whistler*).

A successor to *[HoriWebsite](HoriWebsite)* (and its cancelled successor tentatives *[LexSite](LexSite)*, *[NovaSite](NovaSite)*, *[NekoSite](NekoSite)* and *[Nishi Blade](Nishi_Blade)*), development officially began on 9 February 2026 with the compilation of [build 1](Cuber_build_1). Similar to [Axeon Network](Axeon_Network)'s [Deltari](Deltari) (and [its](KuroWiki) [forks](Deltari_Meta)), *Cuber* uses the *[Whidbey](Whidbey)* and *[Panther](Panther)* development tools, thus it uses the same version format as post-Windows 2000 Microsoft Windows builds, similar to *Deltari*.

Along with the public release of what would eventually become *Cuber*, a variant of it, codenamed ***CuberSH*** (short for *Cuber Self-Hosted*), would serve as a public, modular version of the CSS framework used in *Cuber*, and it is planned to release alongside *Cuber*.

Nevertheless, the development of *Cuber* has been slow, since it has been worked on alongside *[KuroWiki](KuroWiki)* and *[AstroNT](AstroNT)*, which is visible on the time differences between many builds.
