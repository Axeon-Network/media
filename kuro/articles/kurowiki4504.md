**KuroWiki build 4504** is a Release Candidate 1 build of [KuroWiki](KuroWiki). It was compiled on February 15 2026 at 8:40pm.

# Background
This is the first build compiled from the back-then newly made `mdelta_N` branch, which serves as a place to merge the code from the development branches (`kitsixtyfour` and `averyeclipse`) similar to the [four](KuroWiki_build_3615) [Interim](KuroWiki_build_3680) [Developer](KuroWiki_build_3810) [Releases](KuroWiki_build_4100) (IDRs) of KuroWiki. However, unlike the IDRs, builds compiled from this branch are not publicly released, eliminating the need or IDRs entirely.
