**Cuber build 287** is a Developer Preview build of *[Project Cuber](Cuber)*. It was compiled on 19 March 2026 at 4:22 pm. This build is notable for incorrectly having the build lab of `Microsoft Windows [Version 10.0.14393]`, which is the result of a *[Panther](Panther)* bug, where it used the output of the `ver` command on Windows 10 Anniversary Update as the build lab. Due to its incorrect lab, the actual lab of this build remains unknown.
