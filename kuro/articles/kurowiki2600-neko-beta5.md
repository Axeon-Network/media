**KuroWiki build 2600 (nekori64.250927-1322)** is a Beta 5 Escrow build of [KuroWiki](KuroWiki). It was compiled on September 27, 2025 at 1:22 pm UTC-6 from the `nekori64` development branch.

This build is an earlier compile of [the official Beta 5 release](KuroWiki_build_2600_(main)), as it was compiled one day earlier. This build is essentially identical as the aforementioned build, however, [Contributing to KuroWiki](Contributing_to_KuroWiki) is a leftover from Beta 3 due to that it identifies itself as "Contributing to the LexWiki". According to [commit 8b2637e](https://github.com/Axeon-Network/kurowiki/commit/8b2637ed5cf948db6dbf98dd0de8e42b2ba75144), this was fixed in a recompile of this build before the public release of Beta 5. Additionally, this build does not have any code changes from the `kayaurora` development branch, which is notable by many mentions of "Lexibyte" instead of "Nekori" ([Kit](KitSixtyFour)'s alises on July and August 2025 respectively).

This build is available and can be run in the current date by going to [github.com/Axeon-Network/kurowiki/tree/4916310d0b611f258e9ea281e57698eba72545cf](https://github.com/Axeon-Network/kurowiki/tree/4916310d0b611f258e9ea281e57698eba72545cf), which is a snapshot of commit `4916310`. However, in this build (and many other KuroWiki builds prior to [4450](KuroWiki_build_4450)), any references of "getmdl.io" must be changed to "axeon-network.github.io/kurowiki/res/material" for Material Design Lite to work correctly, as getmdl.io was taken down by Google in January 2026.
