**KuroWiki build 2600 (main)** is the first and official Beta 5 build of [KuroWiki](KuroWiki). It was compiled and released on 28 September 2025 by the [Axeon Network](Axeon_Network).

This build jokingly identifies itself as "KuroWikiXP" (albeit "XP" being in orange) since its build number is identical to [the RTM build of Microsoft Windows XP](https://betawiki.net/wiki/Windows_XP_build_2600).

This build is the first public build released under the Axeon Network umbrella, as [Beta 2.1](KuroWiki_build_602) from June 2025 identified itself as "HoriWiki". However, it is not [the first build to identify itself as "KuroWiki"](KuroWiki_build_2000).

# Bugs and quirks
- Search uses an older version of the search index (`search.json`), causing modern pages to not show up in the search results.
- The SpringViewer version watermark is left enabled.
