**Minimal Cuber**, commonly shortened to **MinCuber** or **MinCbr** and formally **Project Cuber, Minimal Edition**. is an upcoming personal website intended to be a replacement to the *[HoriWebsite](HoriWebsite)*, similar to the parent project *[Project Cuber](Cuber)*. Although it serves the same purpose as Cuber, it can be considered a "Demo Version" of Cuber, since it builds upon an preliminary version of the Alpha milestone as a public website. It was made as a response to a quicker replacement of the HoriWebsite, which is outdated and rusty.
