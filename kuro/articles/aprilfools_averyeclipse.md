[April Fools](https://en.wikipedia.org/wiki/April_Fools'_Day) is an annual tradition marked on the first day of April. In the [AveryEclipse](AveryEclipse) universe, the first year April Fools was celebrated was in 2018, and has continued to be observed every year to this day, except for 2020.

# 2026
In 2026, April Fools was marked by a name change of the [Axeon Network](Axeon_Network#Discord_Server) Discord Server to "Akshion Netwok" with a "distorted-looking" server icon made in Microsoft Paint. An "supah megea ultrea admon" role was created to which any non-admin could apply on request, which however did not have any special permissions like the real Admin.

# 2025
The LynxPlaza Discord Server was renamed to "Axeon" with the early 2019 server icon as a "return to roots". That said, however, after 3 weeks the LynxPlaza server has actually been renamed Axeon Network for good, following phase out of frequent videos on the [Lynxmic](Lynxmic) channel.

On the Lynxmic channel, the video "*New windows 7 pre beta - first look*" was uploaded as a "follow-up" to the previous year's April Fools video, centred around Windows 7 Build 7000, recorded entirely with an unregistered HyperCam 2 and edited with an unofficial port of Windows XP's Movie Maker to Windows 7.

# 2024
The Lynxmic channel saw an upload titled "*How to get unregistered hypercam 2*" recorded with an unregistered HyperCam 2 and edited with Windows Movie Maker, entirely inside a Windows XP virtual machine, following the style of late 2000s tutorials using Notepad.
